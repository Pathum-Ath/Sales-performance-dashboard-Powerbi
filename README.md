## 📊 Project Overview
This dashboard was developed to explore sales performance and uncover insights related to revenue, products, and customer behavior.

## 📈 Key Features
- KPI cards for Total Sales, Profit, and Orders
- Regional performance comparison
- Product category and sub-category analysis
- Customer segmentation insights
- Interactive filtering (Region, Category, Date)

## 💡 Key Insights
- Certain regions consistently generate higher revenue
- Technology products contribute significantly to overall sales
- Customer segments show distinct purchasing patterns
